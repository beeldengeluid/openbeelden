Howto

  • Enable this plugin on all HTML5 video or audio tags in your page, for example in its main column <div class="main" /> of your website with: 
    $(document).ready(function() {
    $('div.main').oiplayer({
        'server' : 'http://192.168.1.4:8080/', /* msie (or windows java) has issues with just a dir */
        'jar' : 'plugins/cortado-ovt-stripped-wm_r38710.jar',
        'flash' : 'plugins/flowplayer-3.1.1.swf',
        'controls' : true
    });
});
  • Make the fallback mechanism work by installing Flowplayer and Cortado in the plugins directory. Flowplayer needs the Flash plugin and plays h.264, flv and mp4 files when they are correctly encoded. Cortado is a Java applet that plays Ogg video and audio (ogv, aga).
